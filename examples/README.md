# Örnek Projeler (Examples)

Bu dizin, VCF metodolojisi ile inşa edilmiş gerçek referans projelerini barındırır.
Teori burada pratiğe dönüşür.

## Referans Projeler

- **NuminOS:** Yapay zeka yerleşik (AI-native) Arch Linux dağıtımı.
- **Multiplayer Coding Duel:** Gerçek zamanlı rekabetçi kodlama platformu (React + Socket.io + Judge0 + Redis).

Buraya eklenecek her proje, VCF'in gücünü ve uygulanabilirliğini kanıtlamalıdır.
Kavramsal kod parçacıkları değil, çalışan projeler ekleyin.
